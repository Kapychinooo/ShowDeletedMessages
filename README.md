# ShowDeletedMessages

Плагин для [BetterDiscord](https://betterdiscord.app) который показывает удалённые сообщения прямо в чате.

---

## Что умеет

- Показывает удалённые сообщения с пометкой **УДАЛЕНО**
- Отображает текст, картинки и вложения из удалённого сообщения
- Показывает аватарку и ник автора
- Сообщения остаются видны после выхода и возврата в канал
- Работает с массовым удалением сообщений (bulk delete)

## Скриншот

> Удалённое сообщение выглядит так:

```
┌─ [аватар] Kakao<3 🌀 друн 👑  2:55  [ УДАЛЕНО ]
│  текст удалённого сообщения
│  🖼️ (картинка если была)
```

---

## Установка

1. Убедись что установлен [BetterDiscord](https://betterdiscord.app) версии **1.9.0+**
2. Скачай файл `ShowDeletedMessages.plugin.js`
3. Открой папку плагинов одним из способов:
   - В Discord: **Настройки → Plugins → Открыть папку плагинов**
   - Или вручную: `%appdata%\BetterDiscord\plugins\`
4. Скопируй файл в папку плагинов
5. В Discord зайди в **Настройки → Plugins** и включи **ShowDeletedMessages**

## Ограничения

- Сообщения кэшируются **только в оперативной памяти** — после перезапуска Discord история удалённых сообщений очищается
- Плагин видит только те сообщения, которые были загружены в чат **пока он был открыт**. Если сообщение удалили пока ты был в другом канале — оно не сохранится
- Не работает в каналах к которым у тебя нет доступа

---

## Как это работает

Плагин перехватывает события Flux Dispatcher в Discord:
- `MESSAGE_CREATE` / `MESSAGE_UPDATE` — сохраняет сообщения в кэш
- `MESSAGE_DELETE` — до того как Discord убирает сообщение из DOM, захватывает данные и вставляет собственный элемент на то же место
- `LOAD_MESSAGES_SUCCESS` — при возврате в канал восстанавливает удалённые сообщения из кэша
